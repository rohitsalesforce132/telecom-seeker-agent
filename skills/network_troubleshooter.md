# Network Troubleshooter

## When to Use
Troubleshooting 5G core network service failures, connectivity issues, and signaling problems.

## Instructions
1. Identify the affected service (AMF, SMF, UPF, etc.)
2. Use graph traversal to map affected interfaces
3. Check runbooks for the specific error type
4. Trace the signal path to identify where the failure occurs
5. Recommend specific remediation steps

## Key Services to Know
- AMF: Registration, mobility management
- SMF: Session management, UPF control
- UPF: User plane forwarding
- gNB: Radio access
