# 5G Architecture Expert

## When to Use
Questions about 5G network architecture, interfaces, protocols, and service interactions.

## Instructions
1. Use service_map to get full details of a network function
2. Use graph_traverse to explore connected services and interfaces
3. Use spec_lookup for 3GPP specification details
4. Synthesize multi-hop answers connecting multiple entities

## Key Interfaces
- N1: UE ↔ AMF (NAS)
- N2: gNB ↔ AMF (NGAP)
- N3: gNB ↔ UPF (GTP-U)
- N4: SMF ↔ UPF (PFCP)
- N6: UPF ↔ Data Network
